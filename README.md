# madmath — NFL Analytics Engine

> Originally authored by [@nlugert](https://github.com/nlugert) as a Java Math Library for sports data analysis. Built with love, and taught to his kid who couldn't learn to code at the time — but turned out fine.

## What started here

A clean, well-structured Java utility library covering:

- Integer and double **sum / average** from lists and arrays
- **Rounding** (ROUND_UP / ROUND_DOWN) to arbitrary decimal places  
- **Squared** and **cubed** operations
- Physical constants: `INCH_CM = 2.54`, `FEET_METER = 0.3048`, `SPEED_OF_LIGHT = 299792458`

## What it became — MadMath Pro

Built on top of @nlugert's original library, **MadMath Pro** is a full NFL Analytics web app:

- 🏈 **4th Down Decision Analyzer** — real analytics model for go/punt/field goal decisions
- 📊 **Stats Engine** — full statistical analysis with distribution charts and leaderboard
- 🔄 **Unit Converter** — powered by the original MadMath constants
- 🧮 **Mad Calculator** — square, cube, sqrt, rounding, power, history
